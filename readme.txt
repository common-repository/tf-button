Add the new Twitter Follow Button to your website to increase engagement and create a lasting connection with your audience.

== Description ==

Adds a Twitter Follow Button widget in your sidebar.

Available in English, French and Dutch!

== Installation ==

1. Upload `tf-button.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure widget
4. Activate widget on your website
